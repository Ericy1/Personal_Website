My Experience at NASA

This summer I will be working with NASA to design and test software for an novel, optical communications terminal that will communicate with spacecraft from LEO to lunar orbit. I'll be using Python and HTTP sockets to calibrate/configure the ground terminal, Docker to support orchestration of software components, RabbitMQ for software interfacing, and a variety of frameworks to design a user configurable ground station UI.
